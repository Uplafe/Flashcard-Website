About FlashForge

A beautiful, open-source flashcard platform built for serious learners. Free forever, no ads, no limits.

What is FlashForge?

Built to make learning stick

FlashForge is a modern flashcard web application designed to help you study smarter. Create rich flashcard decks with text and images, study with an interactive flip-card interface, and export your decks as polished PDFs or portable JSON files.

Sign up for a free account to sync your decks across devices, or use the app right away as a guest — your data is stored locally and travels with you via JSON export.

Setup Guide

Setting up Firebase (optional)

FlashForge works out of the box with local storage — no setup needed. To enable cloud sync, Google sign-in, and public deck sharing, connect Firebase in about 5 minutes:

1

Create a Firebase project

Go to console.firebase.google.com → New Project. Give it a name and follow the prompts.

2

Add a Web App

In your project → Project Settings → Add app → Web. Copy the firebaseConfig object.

3

Enable Authentication

Firebase Console → Authentication → Sign-in method → Enable Email/Password and Google.

4

Enable Firestore

Firebase Console → Firestore Database → Create database → Start in production mode → choose a region.

5

Set Firestore Rules

In Firestore → Rules, paste the rules from README.md to allow users to read/write their own decks and read public ones.

6

Paste your config

Open js/data.js and replace the FIREBASE_CONFIG object with your copied config. Done!

FAQ

Frequently asked questions

Do I need an account to use FlashForge? ▼
No — you can create and study decks as a guest. Your decks are saved in your browser's local storage. Sign in to sync across devices and publish public decks.
Is FlashForge free? ▼
Yes, completely free. No premium tier, no ads, no limits on decks or cards. If you self-host with Firebase, you stay within their free tier for typical usage.
How do I back up my decks? ▼
Use the Export → Download JSON button on any deck. This creates a portable .json file you can re-import at any time, even on a different device or install.
How does the PDF export work? ▼
PDFs are generated in landscape A4 format. Each card gets two pages — the first shows the question, the second shows the answer. A cover page with the deck title is included. Great for printing!
Can I add images to my cards? ▼
Yes! When creating or editing a card, click "Add image" on either the front or back panel. Images are stored as base64 data inside the deck JSON, so they travel with your export.
What keyboard shortcuts are available? ▼
In study mode: Space or F to flip a card, → to mark as known / go next, ← to mark as missed / go previous, R to restart the session, S to shuffle cards. In the card editor: Ctrl+Enter to add a card.
